# 💰 Finance Control API

API REST desenvolvida com Flask para controle financeiro de usuários.

## 🚀 Tecnologias

- Python
- Flask
- Flask-JWT-Extended
- Flask-SQLAlchemy
- SQLite

## 🔐 Funcionalidades

- Registro de usuário
- Login com autenticação JWT
- Rota protegida
- Criar transações
- Listar transações

## ▶ Como rodar o projeto

```bash
git clone <url-do-repo>
cd controle_financeiro
python -m venv .venv
pip install -r requirements.txt
python run.py
